Tidal - Browser Game

Created in 8 hours for the EGX Rezzed Creative Assembly Game Jam. Developed using pure Javascript.

Overview


Platform: Web

Language: Javascript

Tech: -

Team Size: 5

Created: 2016

A paper plane in a storm. How far does it fly? ...Storms are unpredictable.

Created with a randomly assembled group for the EGX Rezzed Creative Assembly Game Jam on 08/04/16. We developed it using Javascript to cater the skills of the team and make it easily playable online.

The team consisted of: 2 artists, Benita and Bryn. 2 programmers, Kyle and myself. 1 audio tech, George.
